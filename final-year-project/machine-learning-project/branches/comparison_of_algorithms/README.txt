To run any of the notebooks make sure Jupyter and Anaconda are installed

How to install can be found here: https://jupyter.org/install

The results are visible in the main_project_notebook without running the code.

You will need the following python modules installed if you want to run the code, 
sklearn
numpy
matplotlib
cv2
fnmatch
imutils

All of which install with pip install

To see all image data graphs and scoring metrics obtained run notebooks/main_project_notebook

All testing notebooks are found in the tests folder

All algorithms programmed can be seen in the algorithms folder
